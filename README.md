# Amazon Sales Analysis

## Project Overview
Analyzed Amazon India sales data with 
1,21,149 records using MySQL.

# Tools Used
- MySQL - Data Analysis
- Excel - Data Visualization

## Key Findings
- Total Revenue: Rs.78,574,007
- Best Selling Category: Set
- Top City: Bengaluru
- Top State: Maharashtra
- Average Order value: Rs. 684
